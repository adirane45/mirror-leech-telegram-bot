## üìã Quick Command List

**Download:**
```
/mirror <link>          - Mirror to Drive
/leech <id>             - Leech from Drive
/qmirror <link>         - Mirror with qBittorrent
/jdmirror <link>        - Mirror with JDownloader
```

**Queue:**
```
/queue                  - Show downloads
/pqueue <gid>           - Pause
/rqueue <gid>           - Resume
/prqueue <gid> <pri>    - Set priority
/cancel <gid>           - Cancel
```

**Files:**
```
/zip <path>             - Create archive
/unzip <path>           - Extract archive
/mediainfo <path>       - Media info
```

**System:**
```
/stats                  - System stats
/ping                   - Bot latency
/speed                  - Speed test
/dashboard              - Web interface
/help                   - All commands
```

For complete list, see USAGE_GUIDE.md or QUICK_REFERENCE.md
